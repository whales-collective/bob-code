# Principe Fondamental

## Un assistant de code n'est pas magique

- Fonctionne comme un développeur face à son chef de projet
- Va essayer de prendre des raccourcis
- Peut faire ce qui l'arrange plutôt que ce qui est demandé
- Son objectif principal : **éviter les erreurs de compilation**

> ⚠️ Résultat : un code qui compile mais avec des compromis silencieux
