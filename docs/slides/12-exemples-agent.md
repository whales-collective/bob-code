# Qu'est-ce qu'un Agent ?

## Définition

Un **agent** est un processus autonome spécialisé qui exécute des tâches complexes multi-étapes.

### Exemples d'agents
- `feature-dev:code-explorer` - Exploration approfondie de codebase
- `feature-dev:code-architect` - Design d'architecture
- `test-runner` - Exécution et debug de tests
