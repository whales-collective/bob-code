# Workflow par Feature

## Principe : Raisonnez en petites features

❌ **Ne jamais demander de créer l'application complète d'un coup**

✅ **Procéder par petites features**
- Créer des feature branches
- Ne pas tout jeter si la dernière feature échoue
- Garder ce qui fonctionne

> Une application se construit brique par brique, pas en un seul bloc
