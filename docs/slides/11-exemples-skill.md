# Qu'est-ce qu'une Skill ?

## Définition

Une **skill** est une commande spécialisée réutilisable dans Claude Code.

### Exemples de skills
- `/commit` - Crée un commit git structuré
- `/review-pr` - Revue de pull request
- `/pdf` - Génère un PDF depuis markdown
