# Workflow de l'Agent Methodical

## Phase 1 : Analyse et Planification
- Questions de clarification (framework, DB, contraintes)
- Génération du plan détaillé par features
- Validation du plan avec l'utilisateur

## Phase 2 : Implémentation Guidée
- Exécution feature par feature
- Demande de confirmation avant chaque étape
- Rapport après chaque feature
