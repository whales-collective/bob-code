# Processus pour Chaque Feature

## Étape 1 : Lister et Spécifier
1. Lister toutes les tâches + Créer une spécification détaillée
2. Créer des issues GitHub (pour les branches "features")

## Étape 2 : Demander un Plan
- Plan détaillé par Claude + Liste des étapes d'implémentation
- Fichiers à modifier/créer

## Étape 3 : Workflow Git
1. **Issue GitHub** → 2. **Feature branch** → 3. **Commits réguliers** → 4. **Pull Request**
